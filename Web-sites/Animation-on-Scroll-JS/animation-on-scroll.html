Animation on scroll com JS

Animation on scroll (Animação ao rolar) em JavaScript é uma técnica poderosa para criar uma experiência de usuário mais dinâmica e envolvente. Ela consiste em disparar animações CSS em elementos HTML apenas quando eles entram na área visível da tela do navegador.

Meu Estudo:

Estudei sobre Animation On Scroll com JavaScript e descobri que realizar estas animações ao rolar nem erá tão dificil quanto pensava ao ver o código pela primeira vez, analisalisei o código e entendi aos poucos cada etapa do processo e seu motivo.

Código Nescessário

const myObserver = new IntersectionObserve((entradas) => {
    console.log(entradas)
    entradas.forEach((entrada) => {
        if (entrada.isIntersecting) {
            entrada.target.classList.remove('hidden');
            entrada.target.classList.add('show');
        } else {
            entrada.target.classList.remove('show');
            entrada.target.classList.add('hidden');
        }
    })
})

const elements = document.querySelectorAll('.hidden');

elements.forEach((element) => myObserver.observe(element));
